# ArcGISProShadowTool
- Grafický stín pro 2D vrstvu, z vrstvy se vytvoří vrstva stínu. Vytvořeno pro školní projekt, aby měla výsledná grafika lepší estetickou kvalitu.
  
## Software
- ArcGIS Pro toolbox (.atbx)
- Testováno na ArcGIS Pro 3.3.1
